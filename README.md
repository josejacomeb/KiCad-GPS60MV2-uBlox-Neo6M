# KiCad-GPS60MV2-uBlox-Neo6M

KiCad >= 9 footprint for the uBlox Neo6M module

## Instructions

1. Download the GY-NEO6MV2 by Yancho Zapryanov from the [Link](https://grabcad.com/library/gy-neo6mv2-2)
2. Open the `step` file with for example with FreeCAD and delete the Antenna part
3. Export to resulting step file with the same time
